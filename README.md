# Machine Learning Fundamentals
Coursework completed as part of the BU Master of Data Science program, covering core machine learning concepts and techniques using Python and sci-kit learn.

## Topics Covered

- Linear Regression
- Generalization and Cross-Validation
- Sampling and Cross-Validation
- Feature Selection
- Decision Trees
- Bagging and Random Forests
- Gradient Boosting
- Classification

## Tools & Libraries
Python, sci-kit learn, pandas, matplotlib, seaborn, Jupyer Notebook

## Structure

- 'Homework_01.ipynb' through 'Homework_08.ipynb' - Weekly assignments applying ML techniques to real datasets
- 'Milestone_01-3.ipynb' - Group project milestone 1, Part 3 (Jonaire Tate)
- 'Milestone_02-2.ipynb' - Group project milestone 2, Parts 1 and 2 (Jonaire Tate)
- 'Final_Project_Report.ipynb' - Technical report and ML pipeline for residential property value prediction using the Zillow Prize dataset. Sections 1 (Executive Summary), 2 (Introduction), 3 (Data Description), and 6 (Conclusion) by Jonaire Tate - coming soon

## 
- Ames Housing Dataset - used in homework assignments for regression and model selection tasks
- Zillow Prize Dataset (Kaggle, 2017) - used in group project milestones and final report; ~77,400 residential property records with 54 features, predicting assessed tax value
